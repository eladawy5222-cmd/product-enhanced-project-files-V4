# tourism seo playbook

Use this reference for seo plans, keyword mapping, content strategy, search console analysis, technical seo, on-page optimization, and local search work for tourism websites.

## seo workflow

1. define the business goal: bookings, leads, organic growth, local visibility, destination authority, or product sales.
2. define target markets and languages: arabic, english, or both.
3. group keywords by intent:
   - commercial: book dahab trip, egypt day tours, cairo weekend trips
   - informational: best places to visit in siwa, what to pack for desert safari
   - navigational: brand or specific tour name
   - local: travel agency near me, cairo tour company
4. map each keyword group to a page type: homepage, category, destination page, tour page, article, landing page, faq.
5. review competitors and current search results when live search is available.
6. optimize page structure: title, meta description, h1, h2s, intro, faqs, internal links, image alt text, cta.
7. build topic clusters around destinations, trip types, seasons, and traveler needs.
8. use search console data to find pages with high impressions and low ctr, queries ranking near page one, and pages losing traffic.
9. recommend technical fixes: crawlability, indexation, canonical tags, internal links, page speed, mobile usability, structured data, broken links.
10. define measurement: clicks, impressions, ctr, average position, organic leads, assisted conversions, indexed pages, revenue/bookings.

## tourism keyword cluster examples

- destination cluster: siwa oasis tours, siwa travel guide, siwa itinerary, siwa desert safari, siwa hotels, siwa from cairo.
- trip type cluster: day tours, weekend trips, private tours, family trips, corporate trips, school trips.
- season cluster: eid trips, summer trips, winter desert tours, christmas/new year packages.
- audience cluster: family-friendly trips, couples trips, budget group tours, luxury private tours.
- problem/need cluster: safe trips for families, how to book domestic trips, best weekend getaway from cairo.

## page-level seo template

For each priority page, provide:

- target keyword and secondary keywords
- search intent
- recommended url slug
- meta title under typical search display limits
- meta description with value proposition and cta
- h1
- h2 outline
- content gaps to cover
- internal links in and out
- image recommendations and alt text
- faq questions
- schema suggestions to verify before implementation
- conversion cta

## bilingual seo rules

- Do not mix arabic and english randomly on the same page unless the brand strategy requires it.
- Prefer separate localized pages for arabic and english when both are important for organic search.
- Localize keywords rather than translating literally.
- Use hreflang guidance only when the site has distinct language versions and the implementation can be verified.
- Arabic meta titles and descriptions should sound natural and specific, not machine-translated.

## structured data guidance

Suggest structured data only when it matches visible page content and can be validated. Common candidates include organization, local business, breadcrumb, faq, product/offer, review, article, and tourist/trip-related schema when currently supported and appropriate. Tell the user to validate final markup before publishing.

## search console analysis prompts

When search console data is available, look for:

- high impressions, low ctr: improve title/meta and align with intent.
- average position 4-15: improve content depth, internal links, faq, and freshness.
- declining clicks: compare date ranges and check seasonality, ranking drops, or competitor changes.
- queries not matched by current pages: create new landing pages or articles.
- pages ranking for wrong intent: split or refocus content.

## seo deliverable checklist

A strong tourism seo answer should include:

- business objective
- assumptions or data sources
- keyword clusters
- page map
- priority actions
- content calendar or backlog
- technical recommendations
- measurement plan
- next 30/60/90-day roadmap when requested
