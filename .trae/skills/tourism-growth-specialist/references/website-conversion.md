# website and conversion playbook

Use this reference for tourism websites, landing pages, trip pages, destination pages, homepage copy, website structure, ux recommendations, and conversion optimization.

## recommended tourism website architecture

- home
- tours or trips listing page
- tour detail page template
- destinations pages
- domestic tours category page
- seasonal offers page
- corporate/group travel page
- about us
- reviews/testimonials
- blog/travel guide
- contact/whatsapp booking page
- faq
- terms, cancellation, privacy, and payment information

## homepage section structure

1. hero: destination promise, core value proposition, main cta, trust cue.
2. featured trips: top domestic trips with price/from, duration, dates, cta.
3. why choose us: safety, licensed operation, local guides, transparent pricing, support.
4. popular destinations: destination cards linked to seo pages.
5. how booking works: choose trip, check availability, pay/reserve, travel.
6. reviews and real photos.
7. seasonal offers or confirmed departures.
8. faq focused on booking, cancellation, transport, payment, and support.
9. final cta: whatsapp, form, phone, or request custom trip.

## tour detail page structure

Use this structure for every program/trip page:

1. seo-friendly h1: destination + trip type + duration.
2. hero summary: short emotional promise + practical trip facts.
3. quick facts: duration, destination, group/private, start point, available dates, price/from.
4. cta block: check availability / book via whatsapp / request itinerary.
5. itinerary: day-by-day or hour-by-hour.
6. inclusions and exclusions.
7. who this trip is best for.
8. accommodation/transport details if applicable.
9. gallery or real photos guidance.
10. reviews or social proof.
11. faq.
12. related trips and internal links.
13. final cta.

## landing page structure

For paid campaigns, keep the page focused on one offer and one conversion goal:

- headline that matches the ad promise
- subheadline with destination, duration, and benefit
- short lead form or whatsapp cta above the fold
- trip highlights
- itinerary summary
- inclusions/exclusions
- price/from or quote request
- trust signals
- testimonials/photos
- faq
- repeated cta

## ux and conversion rules

- Put whatsapp/call cta in the header on mobile.
- Keep forms short: name, phone/whatsapp, destination, preferred date, number of travelers.
- Make prices transparent when possible. If variable, use starting price or request quote.
- Show clear inclusions before asking for payment.
- Use real photos when possible; avoid generic stock photos for trust-heavy pages.
- Add sticky cta on mobile for trip pages and landing pages.
- Keep booking steps visible and simple.
- Add faq close to cta to reduce hesitation.
- Add internal links from blog articles to relevant tour pages.

## page deliverable checklist

When producing a website or landing page deliverable, include:

- page goal
- target audience
- page sections/wireframe
- bilingual copy if requested
- seo meta title and meta description
- h1/h2 suggestions
- ctas
- trust signals
- form fields
- internal link suggestions
- implementation notes for designer/developer
